カーソル移動  

|key|command|
|-|-|
|ctrl + a|カーソルを行の先頭へ移動|
|ctrl + e|カーソルを行の末尾へ移動|
|ctrl + f|カーソルを右へ移動|
|ctrl + b|カーソルを左へ移動|
|alt + b|カーソルを前の単語へ移動|
|alt + f|カーソルを次の単語へ移動|
|ctrl + l|clearコマンドの実行|

履歴

|key|command|
|-|-|
|ctrl + p ctrl + n|コマンド履歴の表示|
|ctrl + r|コマンド履歴のインクリメンタルサーチ|
|alt + p|コマンド履歴の検索|

削除  

|key|command|
|-|-|
|ctrl + d|カーソル位置の文字を削除|
|ctrl + h|カーソル位置の前の文字を削除|
|ctrl + k|行末まで削除|
|ctrl + u|行頭まで削除|
|alt + d|次の単語削除|
|ctrl + w|前の単語削除|
|ctrl + y|貼り付け|
|ctrl + \_|アンドゥ|

その他

|key|command|
|-|-|
|TAB / ctrl + i|コマンド補完|
